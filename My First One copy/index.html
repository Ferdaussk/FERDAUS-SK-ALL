Inventory Management-PHP Web Application

For Local Server:-

  1. First you have to make local server in your computer.
  2. For, Download xampp or any local server making software from google.
  3. Then install the software on your system and run Apache and MySQL.
  4. To confirm your local server is runed or not (Visit:- localhost/phpmyadmin).
  5. Find the Inventory Management-PHP software file, copy it and find xampp->htdocs also paste it here.
  6. Then go to your browser and search "localhost/phpmyadmin" to create database and then create a database.
  7. Now click import and then Choose File. Go to Inventory Management-PHP software root file. Select the database ims_db.sql file. Then Open it and click on the Import.
  8. Then login with your information.

For Live Server:-

  1. Login cpanel.
  2. For your Inventory Management-PHP software file upload visit here.
  3. Go to MySQL Databases from your domin/hosting cpanel and create a database. For database create visit here.
  4. Select your database click import and then Choose File. Go to Inventory Management-PHP software root folder. Select the database ims_db.sql file. Then Open it and click on the Import.
  5. Again go to your browser and search with your domain name for get a login form.
  6. Then login with your information.
